# Supermarket Logo Assets

These assets are used as small, unmodified supermarket marks inside shopping-list badges.

Usage rules in the app:
- Keep original proportions with `object-contain`.
- Do not recolor, crop, distort, or place oversized.
- Keep a neutral white background inside the badge.
- Fall back to text initials when a reliable local asset is not available.

Current local assets:
- `mercadona.svg`: Wikimedia Commons, `Mercadona.svg`
- `carrefour.svg`: Logo asset mirrored locally from logo-teka reference page
- `lidl.svg`: Wikimedia Commons, `Lidl-Logo.svg`
- `dia.svg`: Wikimedia Commons, `Dia Logo.svg`
- `aldi.svg`: Wikimedia Commons, `Aldi Süd 2017 logo.svg`
- `alcampo.png`: Wikimedia Commons, `Alcampo.png`
- `eroski.svg`: Wikimedia Commons, `Eroski logo.svg`
- `el-corte-ingles.svg`: Wikimedia Commons, `Logo Corte Inglés.svg`

Not included yet:
- Consum: the available Wikimedia file found during setup is an old 1991-2005 logo, so the app keeps the initials fallback until a reliable current asset is added.
